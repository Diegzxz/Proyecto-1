Descripcion: 
Este proyecto es una app de consola en Python donde puedes organizar tu agenda de pelis para la semana. Puedes ver cuál es la más larga, calcular cuánto duran en promedio, buscar las más recientes, ver cuántas son para mayores de 18, cambiar el horario de una peli sin que choque con otra, y decidir si puedes llevar a un amigo según su edad. Todo está dividido en dos archivos: uno con la lógica y otro con los menús que ve el usuario.
